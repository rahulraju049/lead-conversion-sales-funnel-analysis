# Lead Conversion & Sales Funnel Analysis

## 📌 Project Overview
This project focuses on analyzing lead conversion data to identify bottlenecks in the sales funnel, evaluate campaign performance, and improve overall lead conversion rates using Python-based exploratory data analysis (EDA).

The objective is to help businesses optimize marketing strategies, improve sales team efficiency, and maximize revenue through data-driven insights.

---

## 🎯 Business Problem
Organizations often struggle with:
- Low lead conversion rates
- Poor lead quality from marketing channels
- Lack of visibility into sales funnel drop-offs
- Inefficient campaign spending

This project addresses these challenges through data analysis and visualization.

---

## 📂 Dataset Features
The dataset contains:
- Lead ID
- Lead Source
- Lead Stage
- Campaign Name
- Source Type
- Sales Representative
- Call Activity
- Conversion Status

## 🔍 Project Workflow

### 1. Data Cleaning
- Checked missing values

- Removed duplicates

### 2. Exploratory Data Analysis (EDA)
Performed:

- Conversion rate analysis

### 3. Visualization
Created:
- Bar charts
- Pie charts
- Count plots
- Crosstab heatmaps
- Funnel insights charts

---

## 📈 Key Insights
✅ Found **24.5% missing call activity**, indicating follow-up gaps.

✅ Identified **top-performing sales representatives** based on conversion.

✅ Determined **best-performing lead sources** for high-quality leads.

✅ Detected campaigns generating high leads but poor conversions.

✅ Highlighted major sales funnel drop-off points.
